图片大小随着网页大小改变而改变
	
	响应式图片：	
		js拿到窗口大小，以此来设置图片的大小
		css 媒体查询，设置不同的css
		img srcset= 这个属性实现，不同的dpr下，src不同
		
逐步加载图像
	
	1. 统一占位符，在图片没加载出来之前，看到的是占位图片
	2. 使用低质量图像占位符，LQIP，npm install lqip
		
		处理后的图片，是base64 编码
		
	3. 使用svg图片占位符，SQIP, npm install sqip
		
		处理后，是svg图片
